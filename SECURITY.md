# Security Policy

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Use [GitHub's private security advisory](https://github.com/sparkoo/apxeer/security/advisories/new) to report a vulnerability. This keeps the details confidential until a fix is available.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested fix (optional)

We will acknowledge the report and aim to provide a fix or response promptly.
